# AWS CloudWatch Monitoring and Alerts

##  Objective
To monitor EC2 performance and receive alerts for high CPU usage.

##  AWS Services Used
- Amazon CloudWatch
- Amazon EC2
- Amazon SNS

##  Architecture Overview
CloudWatch monitors EC2 metrics and triggers alarms.
SNS sends alert notifications when thresholds are crossed.

##  Steps Performed
1. Enabled EC2 monitoring
2. Created CloudWatch alarm for CPU usage
3. Configured SNS notification
4. Tested alarm conditions

##  Outcome
Successfully received alerts for high CPU utilization.

##  Evidence
Includes CloudWatch alarm screenshots and SNS notifications.
